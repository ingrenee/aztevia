Eazyest Slides is a plugin to beautifully transform your WordPress galleries, or to create new galleries with shortcodes.

== Description ==

Eazyest Slides adds the following gallery viewers to WordPress:

* Responsive Gallery with Thumbnail Carousel.

  Transform WordPress gallery to a responsive image gallery with a thumbnail carousel using Elastislide.
  
  Based on a [Codrops tutorial](http://tympanus.net/codrops/2011/09/20/responsive-image-gallery/) by Mary Lou (Manoela Ilic)
  
  Use the shortcode `[eazyest_responsive_gallery]`
  
* Gallery with Thumbnail Proximity Effect

	Based on a [Codrops tutorial](http://tympanus.net/codrops/2012/01/04/thumbnail-proximity-effect/) by Mary Lou (Manoela Ilic)
	
	Use the shortcode `[eazyest_proximity_effect]`
* Gallery with Fast Hover Slideshow

	Based on a [Codrops tutorial](http://tympanus.net/codrops/2012/05/09/how-to-create-a-fast-hover-slideshow-with-css3/) by Mary Lou (Manoela Ilic)
	
	Use the shortcode `[eazyest_fast_hover]`
* Gallery with Direction Aware Hover effect

  Based on a [Codrops tutorial](http://tympanus.net/codrops/2012/04/09/direction-aware-hover-effect-with-css3-and-jquery/) by Mary Lou (Manoela Ilic)
  
  Use the shortcode `[eazyest_direction_aware]`

All shortcodes use options like WordPress [gallery](http://codex.wordpress.org/Gallery_Shortcode)

See demos at the [Plugin Homepage](http://brimosoft.nl/eazyest/slides/)

[youtube http://www.youtube.com/watch?v=NvOIEyOjZrI]

== Installation ==

1. Install eazyest-slides using the WordPress plugin installer.
2. Enable the slides you want to use in Settings -> Media
3. If you want to transform all your galleries, select one.
4. Insert shortcodes in posts or pages to show your beautiful galleries

== Frequently Asked Questions ==

= Can I have multiple galleries per page? =
Sorry, you can have only one gallery per page.
